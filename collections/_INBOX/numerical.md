# 以下/未満 - 以上/超過

e.g. 3個 **以下** (≧)

|1|2|**❸**|4|5|6|7|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|🍏|🍏|🍏|||||

e.g. 3個 **未満** (>)

|1|2|**❸**|4|5|6|7|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|🍏|🍏||||||

```
「以下」 その数字を 含んで それよりも下
「未満」 その数字を 含まず それよりも下
```


e.g. 3個 **以上**が赤りんご (≦)

|1|2|**❸**|4|5|6|7|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|🍏|🍏|🍎|🍎|🍎|🍎|🍎|

e.g. 3個 **超過**が赤りんご (＜)

|1|2|**❸**|4|5|6|7|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|🍏|🍏|🍏|🍎|🍎|🍎|🍎|
