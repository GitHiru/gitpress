## || countif（）

Excelや、Spreadsheetではお馴染み、`countif()`。

BigQueryでも使えるよ。


```sql
select
    count(if({}, true, false)) as COUNT_HOGE
```

```sql
select 
    countif({}) as COUNT_HOGE
```
